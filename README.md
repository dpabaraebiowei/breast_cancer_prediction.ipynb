# Breast Cancer Prediction Model

## Overview
I'mThis project applies machine learning to classify breast cancer tumors as malignant or benign using clinical diagnostic features.

## Methods
- Data preprocessing
- Train-test split (80/20)
- Logistic Regression model
- Model accuracy evaluation
- Confusion matrix analysis

## Tools
- Python
- pandas
- scikit-learn
- matplotlib

## Results
The model achieved high accuracy in distinguishing malignant from benign tumors, demonstrating the importance of computational approaches in medical diagnostics.

## Significance
This project demonstrates how machine learning can support early cancer detection and clinical decision-making.
